Para a instalação:

make

Para a execução:
./fb3 -a <arquivoSaidaArvore> (opcional) -o <nomeArquivoSaida> <nomePrograma>
./fb3 -a arvore.txt -o resultado programa.calc
./fb3 -o resultado programa.calc

Após os comandos serão gerados:
    resultado.ll
    executável do resultado
Caso tenha a flag -a:
    arvore.txt
